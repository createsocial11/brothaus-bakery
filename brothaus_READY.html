Deploy for Free

4-Day Setup Guide

From zero to a live bakery AI website — completely free using the tools in the guide.

1
Setup Environment (Day 1)
Install Python, VS Code, and create your project. Get a free Gemini API key from aistudio.google.com — no credit card needed.
mkdir german-bakery-ai && cd german-bakery-ai python -m venv venv pip install fastapi uvicorn google-generativeai python-dotenv # Get free API key → aistudio.google.com
2
Build Python Backend (Day 1–2)
FastAPI backend with /forecast, /simulate, /chat, and /recipe endpoints. Auto-generates API docs at /docs.
# backend/main.py from fastapi import FastAPI # 22-factor forecast engine # 6-step Poisson simulation # Gemini AI chat (Klaus) uvicorn main:app --reload # Visit: http://localhost:8000/docs
3
Connect Frontend (Day 2–3)
This HTML file is your complete frontend. Update the API_URL variable to point to your backend — local or deployed.
// In script section of index.html: const API_URL = "http://localhost:8000"; // For deployed version: const API_URL = "https://your-app.onrender.com"; // All API calls use fetch() — no framework needed
4
Deploy Free on Render + Netlify (Day 3–4)
Push to GitHub. Deploy backend on Render.com (free tier). Drop frontend folder on Netlify — instant live URL. Total cost: ₹0.
# Deploy backend on Render.com (free) git push origin main # Build: pip install -r requirements.txt # Start: uvicorn main:app --host 0.0.0.0 --port 10000 # Add env var: GEMINI_API_KEY=your_key # Frontend: drag & drop folder on netlify.com
